Lab 04 — Token Budget Challenge (3 min)
You have a token budget of 50 input tokens — about 35 to 40 words.
Your goal: get the BEST possible answer about “how to learn Data Structures” using the fewest tokens.

Steps:
Open the OpenAI Tokenizer at platform.openai.com/tokenizer.
Write your prompt, check the token count, then send it to ChatGPT.

Here’s the trick:
Removing filler words like “please”, “can you”, “I would like” saves tokens.
Abbreviations work — “DSA” instead of “Data Structures and Algorithms”.
Format instructions like “table format” improve output without costing many tokens.

Lab 05 — Iterative Conversation (4 min)

You’re going to build a technical architecture for a “Smart Campus Lost & Found” web app using exactly 4 messages.
Message 1: Describe the project and ask AI to question you first.
Message 2: Answer its questions and request a tech stack comparison.
Message 3: Pick an option and ask for detailed architecture.
Message 4: Refine — add notifications, an AI feature, and a free deployment plan.

In 4 messages, you’ll have generated what would take 2 to 3 hours to write from scratch. This is the pattern: Context, Draft, Deepen, Refine.
